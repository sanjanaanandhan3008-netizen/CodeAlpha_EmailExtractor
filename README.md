# CodeAlpha_EmailExtractor

## Project Description
This Python script extracts email addresses from a text file and saves them into a separate file.

## Features
- Reads text from a file
- Extracts email addresses using Regular Expressions (Regex)
- Saves extracted emails to another file
- Displays extracted emails in the terminal

## Files
- email_extractor.py
- sample.txt
- emails.txt

## Technologies Used
- Python
- re (Regular Expressions)
- File Handling

## How to Run
1. Open terminal.
2. Navigate to the project folder.
3. Run:

python email_extractor.py

## Sample Output
support@gmail.com
sales@company.com
admin@yahoo.com
